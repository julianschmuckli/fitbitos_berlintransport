# Berlin Transport for Fitbit OS
This is an app for Fitbit OS to see the next departures for stations in Berlin-Brandenburg.

To build this app yourself, you have to own a Fitbit Ionic and visit this website: https://studio.fitbit.com/

To download the app directly follow this link on your smartphone: https://gam.fitbit.com/gallery/app/9830f043-b9aa-4400-bd5f-618e9d96e76a
